# CS 473 Labs

This repository contains lab work for CS 473. Every lab in `labs/` was completed by hand, and as such was an amazing experience. I learned how to tensorize complex nested loops and the basics of the set complement to deep learning in ML. This class was taught by Dr. Wingate, who provided the best teaching I've ever had.

## Structure

Each week, students participated in two class periods, and submitted two labs. 
Labs are organized by week and type:

- `week_X_main_lab` — main labs (course `cs473` assignments)
- `week_X_mini_lab` — mini-labs (shorter weekly exercises)
