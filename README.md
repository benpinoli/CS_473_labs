# CS 473 Labs

This repository contains lab work for CS 473. Every lab in `labs/` was completed by hand—no automated code generation was used to produce the solutions.

## Structure

Labs are organized by week and type:

- `week_X_main_lab` — main labs (course `cs473` assignments)
- `week_X_mini_lab` — mini-labs (shorter weekly exercises)

Files use the naming convention `week_X_[main/mini]_lab` with either a `.ipynb` or `.py` extension depending on the original format.
